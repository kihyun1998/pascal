
# PASCAL - Professional Analysis of Stocks by Calculated Assessment Logic

투자 의사결정을 위한 크로스플랫폼 주식 밸류에이션 애플리케이션

## 핵심 기능
- PER/PSR 기반 기업가치 분석
- 적정주가 산출 및 고평가/저평가 판단
- 기업 재무정보 시각화

## 지원 플랫폼
- Windows
- macOS
- Linux
- iOS
- Android

## 기술 스택
- Framework: Flutter
- State Management: Riverpod
- Architecture: Clean Architecture + MVVM
- Charts: fl_charts
